BlockUndentN

Key: Ctrl+K U
Menu: Word > Indentation > Undent spaces

Ask for a number and remove that many spaces from the beginning of the block. If the number is greater than the number of spaces on the first non-blank line, use that number instead. See BlockUndent, for a side-effect.

This function is not available for a stream block.
