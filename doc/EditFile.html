EditFile

Key: Shift+F4
Menu: File > Load

Prompt for a name and load the file into the editor. If the name is a subdirectory or contains wildcards then bring up the directory list (see DirList); if the name is `=' the name of the current file will be used. The file will be made the current window, with the previous window becoming hidden.
