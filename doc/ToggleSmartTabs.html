ToggleSmartTabs

Key: Alt+Tab
Key: Shift+Ctrl+I
Menu: Word > Tabs > Smart tabs

Alternate between fixed tabs and smart tabs. Fixed tabs use the logical tab size to determine the next (or previous) tab stop; smart tabs use the next (or previous) string in the previous non-blank line.
