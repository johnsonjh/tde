Quit

Key: F3
Key: Alt+Q
Key: Ctrl+K Q (UNIX)
Menu: Window > Close

Close the current window, prompting to abandon changes. If it is the last window, the next file on the command line will be opened, or the editor will exit if none. If there are multiple windows or files, then if there is a hidden window, it becomes the current; otherwise the window above or below, or to the left or right, will grow to accomodate the closed window; if this is not possible, the window will not be closed.
