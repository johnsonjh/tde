Help

Key: F1
Key: Ctrl+J
Key: Alt+H
Menu: n/a

Display a help screen. The screen can be customised by creating a tde.hlp file (DOS/Windows) or .tdehlp file (UNIX). It will be loaded from the current directory, the directory contained in the TDEHOME or HOME environment variables, or the same directory as the executable, whichever exists first. The file can contain two help screens: one for the normal editor and one for a read-only file; use a `<<<>>>' sequence to separate the two. The screens can be a maximum of 50 lines, with each line having a maximum of 100 characters.
