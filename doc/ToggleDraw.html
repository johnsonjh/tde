ToggleDraw

Key: Alt+Insert
Menu: Toggles > Drawing

Turn drawing mode on or off. In drawing mode, the (Stream)CharLeft/Right and LineUp/Down functions will draw a line (or corner or intersection, as appropriate) using the current graphic set (see ToggleGraphicChars). Turning on draw mode will automatically turn on the graphic characters.
