ContextHelp

Key: Shift+Ctrl+F1
Menu: Help > Context Help

Loads a file into the editor (or switches to it if it is already loaded) and, if the cursor is on a word, searches it for that word (using a regular expression). The default file is tde.txt and the default expression is `^<~>' (where `~' is replaced with the word); these can be changed in the configuration. The file can have a relative path, but if it exists when the editor is started, or once it has been loaded, it will be made absolute.
