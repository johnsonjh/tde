GotoWindow

Key: Ctrl+F11
Menu: Window > Goto

Select a window by number and letter, or filename. If the string starts with a digit, it is assumed to be the file number; if there is no following letter, find the first (visible) window. If the string does not start with a digit, it is assumed to be the start of the filename (the name itself, not the path). If no window is chosen (i.e. an empty string) a window list is displayed. This is similar to the directory list (see DirList). This list uses symbols to indicate the type of window. The first symbol is `!' for the current window, `-' for a hidden window and `+' for a visible window. Then comes the window number and letter and the three file indicators: `!' for read-only, `#' for scratch and `*' for modified. The window title or relative file name is last, which also determines the sorting. Typing a number will select that window number; typing anything else will select the name.
