Issues

Most users should have no problem running TDE, but there a few situations that cause troubles.

TDEW & TZ
If you need to use the TZ environment variable, include the abbreviation, otherwise the current time will be wrong. Eg: use `set TZ=EST-10', not `set TZ=-10'.
TDEP & DR-DOS
The built-in DPMI server needs to be used, otherwise the keyboard will not function correctly.
Cygwin/MSYS
TDE will not run correctly in the shells of these environments. Use start to run TDE in a new console window.
