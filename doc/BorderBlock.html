BorderBlock

Key: Shift+Alt+B
Menu: Block > Border > Standard

Draw a border/frame within the confines of a box block. The characters used to generate the border are configurable (press the Help key for how). This function will always draw each corner and side (use a space to “hide” one) using one character; BorderBlockEx, for a multi-character border, with optional corners and sides.

The graphic characters (see ToggleGraphicChars) are automatically turned on at the start of this function; once completed, they are turned off again (if they were not already on).
