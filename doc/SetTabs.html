SetTabs

Key: Ctrl+Tab
Menu: Word > Tabs > Set tabs

Set the logical and physical tab sizes. The logical tab size determines the number of spaces used when the tab key is pressed (in fixed tab mode); the physical tab size determines the number of spaces used by the actual tab character. For example, setting logical tab size to `4', but physical tab size to `8', would mean Tab would have to be pressed twice to generate one tab character.
