BlockCenterJustify

Key: Shift+Alt+F10
Menu: Word > (Block Justify) Center

Center justify the contents of the block. For a line block, each line is centered according to the current margins; for a box block, text within the block is centered according to the left and right edges of the block; this function is not available for a stream block.
