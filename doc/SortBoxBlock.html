SortBoxBlock

Key: Alt+S
Menu: Block > Sort

Sort the lines in the block, prompting to sort ascending or descending. If a line block is marked, the entire line acts as the key; a box block will use the contents of the block as the key; this function is not available for a stream block. To sort multiple columns, sort the least significant column first. The case mode (see ToggleSearchCase) controls the case sensitivity of the sort.
